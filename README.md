# 🏃‍♂️Stretching Care
-   자세 교정, 스트레칭에 도움을 주는 앱입니다
-   GIF 형식의 이미지 와 자세한 설명을 보며 스트레칭을 해보세요
-   날짜 별로 스트레칭 메모를 저장하고 영상을 시청해 보세요
<h2 id="3">앱 기능 설명</h2>

-   스트레칭 별로 GIF 이미지와 자세한 설명을 확인할 수 있습니다
-   스트레칭을 시작하면 타이머가 시작되고 다음, 이전 스트레칭으로 넘어갈 수 있습니다
-   타이머를 정지할 수 있고 타이머가 끝나기 5초 전에 사운드가 재생됩니다
-   날짜별로 스트레칭 메모를 저장할 수 있고 수정, 삭제할 수 있습니다
-   스트레칭에 도움을 주는 영상을 시청할 수 있습니다
